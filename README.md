A. Create S3 buckets via Terraform
Steps:
* Install AWS CLI
* Configure,and set up AWS KEY and SECRET creds
* Author S3 creation tf

1)Install AWS CLI
curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip" -o "awscliv2.zip"
unzip awscliv2.zip
sudo ./aws/install

2) Run aws configure
